s4-codec-wasm — browser-side S4F2 decoder

This page decodes a hard-coded S4F2 sample (CPU-zstd codec) using the WASM bundle built from crates/s4-codec-wasm. It does not hit S3 — the bytes are inlined as base64 below so the demo works offline.

Build the bundle first: cd crates/s4-codec-wasm && wasm-pack build --release --target web --out-dir pkg, then serve this directory via any static HTTP server (python3 -m http.server works) and open /examples/web-demo/index.html.
